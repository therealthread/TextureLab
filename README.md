# CustomModelData

- **Model Types**
  - SimpleModel
  - Empty
  - Multiple Tint Model (Per model's texture)
    - Tints:
      * Constant
      * GrassTint (temperature, downfall)
      * DyedTint
      * Team (Determining according to the team's colors)
      * Potion
  - Condition Models
    - Simple Condition
    - Keybind
    - Damaged Item 
    - Has Component
    - Component Condition Model
      * EnchantmentEntry
      * EnchantmentTypes
  - CompositeModel
    * Multiple ModelData entries
  - SimpleMatchSelect
  - SimpleMatchRange
