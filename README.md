# TextureLab
A custom Resource Pack generator and viewer for Minecraft: Java Edition. Developed in Java and using JFX (non-modular) libraries, it features an interface built from scratch on a canvas-based render pipeline.

![](https://github.com/therealthread/TextureLab/blob/main/assets/view.png?raw=true)

> [!IMPORTANT]
>
> This application uses textures from the Minecraft game developed by Mojang Studios and
> features a user interface inspired by the Minecraft: Bedrock Edition theme.
